# Humanize writing

Treat the user's provided text as the draft. Rewrite it the way someone who actually lived it would say it: a little rougher, more direct. Cut anything rehearsed, overbuilt, or written to sound impressive. No em-dashes, no "in today's world" preambles, no rule-of-three padding. Vary sentence length. Keep every concrete claim, number, and example. Return only the rewrite. If no text was given, ask for it.
