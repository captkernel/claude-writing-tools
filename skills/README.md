# Skills (for claude.ai and Claude Code)

Agent Skills packaged from this repo's tools. Upload a `dist/*.zip` to
claude.ai (Settings -> Capabilities -> Skills, or a Project), or copy a
skill folder into `~/.claude/skills/` for Claude Code.

- **humanize** — `skills/humanize/SKILL.md` · zip: `skills/dist/humanize.zip`
- **tighten** — `skills/tighten/SKILL.md` · zip: `skills/dist/tighten.zip`
- **hooks** — `skills/hooks/SKILL.md` · zip: `skills/dist/hooks.zip`
